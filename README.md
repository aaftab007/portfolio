- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Python, Java, FastAPI, Node.js
- **ML/AI**: PyTorch, scikit-learn, BERT, Computer Vision
- **Data**: Apache Spark, Kafka, Redis, PostgreSQL
- **Cloud**: AWS, GCP, Docker, Kubernetes
- **Tools**: Git, GitHub Actions, Prometheus, Grafana

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/aaftab007/portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
